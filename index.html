QuizletBot - The Free Alternative

We know you don't want to pay $2.00 for Quicklet, so we have come up with QuizletBot! QuizletBot is dedicated to helping you achieve that unbeatable score- we hope you enjoy it!

Usage and Installation Guide

QuizletBot is incredibly simple and easy to use- just follow the steps below.

  1. Install the plugin
  2. Go to any Quizlet Scatter or Gravity game
  3. Click the QuizletBot icon in the top-right corner (plugin icon)

Scatter Directions:

  1. Match the two cards that say "Match Me" together, I know, I'm working on the automation.

Gravity Directions:

  1. Click the QuizletBot
  2. Make sure the Gravity game is set to the instructions provided
  3. Press Space then Enter to submit

Enjoy!
